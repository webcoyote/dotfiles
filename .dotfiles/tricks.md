# Text selection

Double click a word and to keep the mouse pressed during the second click of the double click. Then moving it while pressed will select whole words. Triple click selects paragraphs and can be hold too.

# Screenshot

# File preview

Space bar to preview everything. No need to wait for apps to open or clutter up the screen. Just select a file hit the space bar. You can even use the arrow keys to scan up and down through the file list.

# Fast web search

Type something into spotlight and hit Command + B and it will perform a web search in your default browser.


# Finder file inspector

Select a file and press Cmd+Option+I (or select File>Show Inspector [Get Info becomes Show Inspector] while holding Option) to get a permanent info window that always shows the info for the currently selected file.

# Finder navigation

To quickly see and navigate the current folder in Finder, Cmd-click the title to get a popup with the folder hierarchy.

# Show hidden files

To show or hide hidden files and folders press the Command (⌘) + Shift (⇧) + Period (.)

# Emoji dialog

Command (⌘) + Control (⌃) + Spacebar

# Zoom

System Settings -> Accessibility -> Zoom -> Use Keyboard Shortcuts to Zoom

Toggle Zoom: Option (⌥) + Command (⌘) + 8
Zoom in: Option (⌥) + Command (⌘) + =
Zoom out: Option (⌥) + Command (⌘) + -
